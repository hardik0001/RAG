# Retrieval-Augmented Generation (RAG) Learning Track

This repository is a hands-on, concept-driven learning track for building Retrieval-Augmented Generation (RAG) systems.

It walks through the complete RAG pipeline from first principles:
- Document loaders
- Text splitters
- Embedding models
- Vector store indexing
- Dense retrieval
- Advanced retrieval techniques:
  - Re-ranking
  - Reciprocal Rank Fusion (RRF)
  - Hybrid search (dense + sparse)
  - Query transformation strategies

The track then expands into production-grade RAG architectures:
- Corrective RAG (CRAG)
- Self-RAG
- HyDE (Hypothetical Document Embeddings)
- Draft-then-RAG (DraftRAG)
- Multi-Modal RAG
- RAG-as-a-Service patterns
- Agentic and guardrailed RAG systems with evaluation and safety layers

Each module is self-contained with implementation examples, making this repository a practical reference for both learning and rapid prototyping.
